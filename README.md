## Array bidimensionales

### Índice y Enunciados:

#### 1. Fundamentos de arrays
- [Ejercicio 1](https://github.com/iescman2002/programacion/blob/practica_3/Practica3/src/Ejercicio1.java)
  
  Define un array de números tipo double de 3 filas por 7 columnas con nombre doub y asigna los valores según la siguiente tabla. Muestra el contenido de todos los elementos del array dispuestos en forma de tabla como se muestra en la figura.
- [Ejercicio 2](https://github.com/iescman2002/programacion/blob/practica_3/Practica3/src/Ejercicio2.java)
  
  Escribe un programa que solicite 20 números enteros. Estos números debemos de introducirlo en un array de 4 filas por 5 columnas. El programa mostrará las sumas parciales de filas y en las columnas el mayor número de la columna. La suma total debe aparecer en la esquina inferior derecha.
- [Ejercicio 3](https://github.com/iescman2002/programacion/blob/practica_3/Practica3/src/Ejercicio3.java)

Modifica el programa anterior de tal forma que los números que se introducen en el array se generen de forma aleatoria (valores entre 1 y 1999).
- [Ejercicio 4](https://github.com/iescman2002/programacion/blob/practica_3/Practica3/src/Ejercicio4.java)

Modifica el programa anterior de tal forma que las sumas parciales y la suma total aparezcan en la pantalla con un pequeño retraso, dando la impresión de que el ordenador se queda “pensando” antes de mostrar los números.
- [Ejercicio 5](https://github.com/iescman2002/programacion/blob/practica_3/Practica3/src/Ejercicio5.java)

Crear un programa que cuando se le introduzca números enteros rellene un array de 6 filas por 10 columnas con números enteros positivos comprendidos entre 0 y 1000 (ambos incluidos). A continuación, el programa deberá:
  <ul>
    <li>dar la posición del número máximo y mínimo</li>
<li>la suma total de todas las filas y columnas.</li>
<li>la suma de todas las columnas.</li>
<li>la suma de todas las filas.</li>
  </ul>
- [Ejercicio 6](https://github.com/iescman2002/programacion/blob/practica_3/Practica3/src/Ejercicio6.java)

Modifica el programa anterior de tal forma que no se repita ningún número en el array además de que tiene que estar comprendido en un rango entre 20-40.
- [Ejercicio 7](https://github.com/iescman2002/programacion/blob/practica_3/Practica3/src/Ejercicio7.java)

Modifica el programa del Ejercicio 6 para que:
<ol><li>Los números NO se repitan (como en el ejercicio anterior).</li>
<li>Los números estén comprendidos en un rango dinámico (el usuario introduce el valor mínimo y máximo).</li>
<li>Al final, el programa muestre:</li>
<ul><li>La media aritmética de todos los números del array.</li>
<li>La posición de todos los números primos que haya en el array.</li>
<li>Una representación gráfica en consola de cada fila, donde cada número se represente con un número de * proporcional a su valor dentro del rango dado (por ejemplo, si el rango es 10-20 y aparece el 15, se mostrarán 5 *).</li></ul></ol>
